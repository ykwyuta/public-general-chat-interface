# チャット機能テストケース (Demo Auth / Gemini 3.1 Flash Lite Preview)

## 目的
ユーザーがデモ認証でログインし、gemini-3.1-flash-lite-previewモデルを選択してチャット機能が正常に動作することを確認する。

## 前提条件
- Next.jsアプリケーションが起動していること。
- `data/users.yaml` にデモ用アカウント（例: `demo`/`demo`）が設定されていること。
- `public/models.yaml` (あるいはシステム設定) に `gemini-3.1-flash-lite-preview` が設定されていること。

## テストステップ

### 1. ログイン処理
1. トップページ（またはログイン画面）にアクセスする。
2. デモ認証（Demo Auth）のログインフォームに移動する。
3. ユーザー名に `demo` 、パスワードに `demo` （または有効なデモユーザーの認証情報）を入力し、ログインボタンをクリックする。
4. **期待される結果:** ログインに成功し、チャットのメイン画面（UI）に遷移すること。

### 2. モデルの選択と設定
1. 画面上のモデル選択ドロップダウンまたは設定(Settings)を開く。
2. プロバイダ/モデル一覧から `gemini-3.1-flash-lite-preview` を選択する。
3. （設定モーダルの場合）設定を保存してモーダルを閉じる。
4. **期待される結果:** チャット画面上で現在選択されているモデルが `gemini-3.1-flash-lite-preview` になっていること。

### 3. メッセージの送信
1. チャット入力欄（テキストエリア）に「こんにちは」と入力する。
2. 送信ボタン（またはEnterキー）を押下する。
3. **期待される結果:**
   - 入力欄がクリアされること。
   - チャット履歴にユーザーのメッセージ「こんにちは」が追加されること。
   - AIの応答待ちインジケーターが表示されること。

### 4. AIからの応答確認
1. AIからの応答がチャット履歴に表示されるのを待つ。
2. **期待される結果:**
   - AIの応答メッセージが正しく表示されること。
   - インジケーターが消えること。
