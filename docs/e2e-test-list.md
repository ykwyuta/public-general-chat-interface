# E2Eテスト一覧

READMEに記載されている機能（F-01〜F-54）に対するPlaywrightのE2Eテスト一覧です。

| 機能ID | 機能 | テストファイル | テストケース概要 |
|---|---|---|---|
| F-01 | メッセージ送信 | `e2e/f01-message-send.spec.ts` | テキスト入力欄からメッセージを送信できる |
| F-02 | メッセージ受信 | `e2e/f02-test.spec.ts` | AI からの返答をストリーミング形式でリアルタイム表示できる |
| F-03 | Markdown レンダリング | `e2e/f03-test.spec.ts` | AI の返答中の Markdown（コードブロック・表・リストなど）を正しく描画する |
| F-04 | コードハイライト | `e2e/f04-test.spec.ts` | コードブロックに構文ハイライトとコピーボタンを提供する |
| F-05 | メッセージ再生成 | `e2e/f05-test.spec.ts` | AI の返答を再生成（リトライ）できる |
| F-06 | メッセージ編集 | `e2e/f06-test.spec.ts` | ユーザーの送信済みメッセージを編集して再送信できる |
| F-07 | 会話リセット | `e2e/f07-test.spec.ts` | 現在の会話をクリアして新規会話を開始できる |
| F-08 | 会話履歴一覧 | `e2e/f08-test.spec.ts` | 過去の会話一覧をサイドバーに表示する |
| F-09 | 会話の新規作成 | `e2e/f09-test.spec.ts` | 「新しいチャット」ボタンから新規会話を開始できる |
| F-10 | 会話の切り替え | `e2e/f10-test.spec.ts` | サイドバーから過去の会話に切り替えられる |
| F-11 | 会話タイトル自動生成 | `e2e/f11-test.spec.ts` | 最初のメッセージを元に会話タイトルを自動生成する |
| F-12 | 会話の削除 | `e2e/f12-test.spec.ts` | 不要な会話を削除できる |
| F-13 | 会話の永続化 | `e2e/f13-test.spec.ts` | 会話履歴を SQLite (better-sqlite3) に保存し、リロード後も復元できる |
| F-14 | モデル選択 | `e2e/f14-test.spec.ts` | 使用する AI モデルを UI から切り替えられる |
| F-15 | システムプロンプト設定 | `e2e/f15-test.spec.ts` | カスタムシステムプロンプトを設定できる |
| F-16 | プロバイダー設定 | `e2e/f16-test.spec.ts` | 複数のAIプロバイダー（Anthropic, Gemini, Bedrock等）から使用するものを選択できる。APIキーはサーバーの環境変数で管理する |
| F-17 | シナリオ選択 | `e2e/f17-test.spec.ts` | ユーザーが用意されたシナリオから会話相手を選択できる |
| F-18 | YAMLシナリオ定義 | `e2e/f18-test.spec.ts` | 応答ルールや状態遷移をYAMLファイルで定義できる |
| F-19 | 選択肢ボタン表示 | `e2e/f19-test.spec.ts` | シナリオの遷移条件に基づく選択肢をボタンとして表示し、ワンクリックで返信できる |
| F-20 | 条件分岐・状態管理 | `e2e/f20-test.spec.ts` | ユーザーの選択に応じてシナリオ状態が遷移し、適切なテキストや選択肢を表示する |
| F-21 | アーティファクト自動検出 | `e2e/f21-test.spec.ts` | AI 返答内のコードブロック（言語タグ付き）をアーティファクトとして自動抽出する |
| F-22 | アーティファクトパネル表示 | `e2e/f22-test.spec.ts` | チャット領域の右側にアーティファクトパネルを表示する。アーティファクトが存在しない場合はパネルを非表示にし、チャット領域を全幅で使用する |
| F-23 | ファイルごとの展開／折りたたみ | `e2e/f23-test.spec.ts` | 複数のアーティファクトがある場合、各ファイルを個別に展開・折りたたみできる |
| F-24 | アーティファクトのコピー | `e2e/f24-test.spec.ts` | 各アーティファクトのコードをクリップボードにコピーできる |
| F-25 | HTML プレビュー | `e2e/f25-test.spec.ts` | `html` 言語のアーティファクトは、コード表示とブラウザプレビュー（iframe）を切り替えられる |
| F-26 | ファイル名の推定・表示 | `e2e/f26-test.spec.ts` | 言語タグやコード内容からファイル名（例: `index.html`, `component.tsx`）を推定して表示する |
| F-27 | アーティファクト一覧ナビゲーション | `e2e/f27-test.spec.ts` | パネル上部に全アーティファクトの一覧（ファイル名リスト）を表示し、クリックでジャンプできる |
| F-28 | メッセージとアーティファクトのリンク | `e2e/f28-test.spec.ts` | どのメッセージのターンで生成されたアーティファクトかを紐づけて管理する |
| F-29 | レスポンシブデザイン | `e2e/f29-test.spec.ts` | PC・タブレット・スマートフォンで適切に表示される |
| F-30 | ダークモード | `e2e/f30-test.spec.ts` | ライト／ダークテーマを切り替えられる |
| F-31 | 送信中インジケーター | `e2e/f31-test.spec.ts` | AI 応答待ち中にローディングアニメーションを表示する |
| F-32 | 入力欄の自動リサイズ | `e2e/f32-test.spec.ts` | テキストエリアが入力量に応じて自動的に縦伸びする |
| F-33 | Shift+Enter で改行 | `e2e/f33-test.spec.ts` | Enter で送信、Shift+Enter で改行する |
| F-34 | スクロール追従 | `e2e/f34-test.spec.ts` | 新しいメッセージが届くと自動スクロールする |
| F-35 | メッセージのコピー | `e2e/f35-test.spec.ts` | メッセージ本文をクリップボードにコピーできる |
| F-36 | タスク作成・管理 | `e2e/f36-test.spec.ts` | タイトル、目的、完了条件を指定してタスクを作成・管理できる |
| F-37 | マルチエージェント | `e2e/f37-test.spec.ts` | 1つのタスクに複数のLLMエージェントと人間が参加し、発言できる |
| F-38 | メンション機能 | `e2e/f38-test.spec.ts` | `@agent_name` の形式で特定のエージェントを指名して発言を促すことができる |
| F-39 | タスクテンプレート | `e2e/f39-test.spec.ts` | よく使うタスク設定（参加エージェント構成含む）をテンプレートとして保存し、再利用できる |
| F-40 | ログイン | `e2e/f40-test.spec.ts` | Google OAuth、および YAMLファイルで定義したデモ用アカウントによるログインが可能 |
| F-41 | ユーザー管理 | `e2e/f41-test.spec.ts` | ログインユーザーごとのチャット履歴やタスク、テンプレートを分離して管理・保存できる |
| F-42 | 画像の添付 | `e2e/f42-test.spec.ts` | ファイル選択により複数の画像をメッセージに添付できる |
| F-43 | 添付画像のプレビューと削除 | `e2e/f43-test.spec.ts` | 送信前に添付した画像のサムネイルをプレビューし、削除できる |
| F-44 | テンプレート作成・管理 | `e2e/f44-test.spec.ts` | ウェルカムメッセージ、システムプロンプト、デフォルトMCP、コンテキストコンテンツを設定して保存できる |
| F-45 | ワークスペースへの自動展開 | `e2e/f45-test.spec.ts` | テンプレートからチャットを開始する際、事前提供データをセッションごとのワークスペースに自動コピーする |
| F-46 | MCPサーバーの接続管理 | `e2e/f46-test.spec.ts` | stdio（ローカル）やSSE（リモート）形式でMCPサーバーを登録・管理・接続できる |
| F-47 | ツールの動的拡張 | `e2e/f47-test.spec.ts` | MCPサーバーが提供するツールをUIから選択し、LLMに公開して実行できる |
| F-48 | リソースのコンテキスト注入 | `e2e/f48-test.spec.ts` | MCPサーバーが提供するデータをメッセージのコンテキストとして添付できる |
| F-49 | ツール実行の可視化 | `e2e/f49-test.spec.ts` | チャット内でMCPツールが実行されたことを専用バッジとして視覚的に表示する |
| F-50 | 組み込みファイル入出力MCP | `e2e/f50-test.spec.ts` | LLMが生成した内容の保存、ワークスペース内ファイルの読み込みを安全に行う |
| F-51 | メンション・アーティファクト指定 | `e2e/f51-test.spec.ts` | `@username` と `#artifact_name` を用いて宛先と対象ファイルを指定してメッセージを送信できる |
| F-52 | アプリ内通知機能 | `e2e/f52-test.spec.ts` | メンションされたユーザーにベルアイコンのバッジで通知し、一覧を表示する |
| F-53 | 依頼チャットの開始 | `e2e/f53-test.spec.ts` | 通知から、共有されたアーティファクトとメッセージを引き継いだ新しいチャットを開始できる |
| F-54 | 依頼事項バナー表示 | `e2e/f54-test.spec.ts` | 依頼チャットの画面上部に「依頼事項」「依頼者」「依頼日時」をバナー表示する |
